## Current directives

_(seeded 2026-07-21 — 04-seo updates this file as evidence accumulates)_

1. **Title pattern:** Lead with "Best {product type}" phrasing for roundups; include the count or
   the head-to-head names ("X vs Y") when natural. Keep titles ≤ 60 characters where possible so
   they don't truncate in SERPs.
2. **Meta description:** 140–165 characters, includes the primary keyword once, states the
   comparison's honest angle (verified weights, sourced specs). No clickbait.
3. **FAQ targeting:** Write FAQ questions in people-also-ask phrasing (natural questions a
   beginner types), 3–5 per article, answers 40–80 words.
4. **Internal links:** Every new article links 3–5 existing articles; prioritize linking INTO
   money pages from guides, and OUT to the two nearest category-mate articles from roundups.
5. **Verdict box discipline:** The three awards must appear within the first screen. Do not bury
   the answer — the verdict box exists so impatient readers convert before scrolling.
6. **Word count:** 1,400–2,200 words for roundups. Guides may run 1,000–1,800.
7. **Keyword density:** Primary keyword in title, H1, first 100 words, exactly one H2, and meta
   description; overall density under 1.5% — write naturally and count afterward, not the reverse.
