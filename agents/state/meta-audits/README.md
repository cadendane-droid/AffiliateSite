# Meta audits

Owned by **07-meta-auditor**. One report per weekly run: `YYYY-MM-DD.md`, append-only (past
audits are never edited — corrections go in the next audit with a reference back).

Each report contains: UTC-timestamped header, the docket (every 05-meta instruction change
since the last audit, reconciled against git history), a four-part judgment per decision
(intent / wisdom-at-the-time / process / hindsight-labeled-separately) with every claim
citing its source (file+date, snapshot filename, commit SHA), a verdict
(SOUND / QUESTIONABLE / UNSOUND / FLAGRANT), systemic observations, and any actions taken
with their own timestamps and resulting commit SHAs.
